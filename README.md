# About
Orderium is a plugin that allows players on your server to create and deliver orders from each other for money and supplies.
<br>It is inspired by [DonutSMP](https://store.donutsmp.net/).
<br>Download the plugin on [Modrinth](https://modrinth.com/plugin/orderium).

# Documentation
Documentation for usage of Orderium can be found in [this repository wiki](https://github.com/ImKarven/Orderium/wiki).

# Compiling
Requirements: JDK 21+
<br>Once the repository is cloned, run `./gradlew shadowJar` to compile the plugin.
<br>Compiled jar file can be found in build/libs
